HTTP Interceptor

Click here to send an Http request that passes through an HTTP interceptor and a caching interceptor.

The HTTP interceptor appends an Authorization header.

The caching interceptor stores cached responses in indexedDB. A response expires in 3 minutes. (See the Network tab in Developer Tools.)

Click here to send an Http request that bypasses the interceptor.

Error Handler

The default implementation of Angular's ErrorHandler prints error messages to the console.

A custom handler intercepts error handling and sends an error to the backend server. Open the DevTools and click the Network tab to see the request that is sent from the error handler.

Error HTTP Interceptor

Error interceptor captures client-side and server-side errors.
